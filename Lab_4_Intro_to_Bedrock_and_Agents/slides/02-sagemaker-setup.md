# SageMaker Setup

## Your Development Environment

Amazon SageMaker Studio provides Jupyter notebooks with:
- Pre-configured Python environments
- Access to AWS services
- GPU instances (optional)

## Setup Steps

1. Navigate to **Amazon SageMaker AI** (not "Amazon SageMaker")
2. Click **Set up for single user**
3. Wait for environment setup (1-2 minutes)
4. Click **Open Studio**
5. Launch **JupyterLab**
6. Create a **labs** folder
7. Clone the workshop repository

## Pro Tip

Use the **Quick start** template - it's sufficient for this lab and starts faster.
