# Amazon Bedrock

## What is Bedrock?

Amazon Bedrock provides access to foundation models through a unified API.

## Available Model Providers

| Provider | Models |
|----------|--------|
| **Anthropic** | Claude (Sonnet, Haiku, Opus) |
| **Meta** | Llama 2, Llama 3 |
| **Amazon** | Titan (Text, Embeddings) |
| **Cohere** | Command, Embed |
| **AI21** | Jurassic |

## Key Concepts

| Concept | Description |
|---------|-------------|
| **Model ID** | Identifies the specific model |
| **Inference Profile** | Cross-region routing for availability |
| **Converse API** | Unified interface for chat models |
