# Lab 7: Neo4j MCP Agent

## The Final Lab

Congratulations on making it to the last lab! Here you'll learn about the **Model Context Protocol (MCP)** and how it enables AI agents to connect to external tools.

## Two Key Concepts

1. **MCP** - An open standard for connecting AI to external tools
2. **Neo4j MCP Server** - Enables natural language queries to graph databases

## What You'll Learn

- What MCP is and why it matters
- How agents discover and use MCP tools
- Natural language to Cypher translation
- Building agents with LangGraph or Strands
