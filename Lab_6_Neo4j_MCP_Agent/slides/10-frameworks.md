# Framework Options

## Two Notebooks, Same Result

| Framework | Notebook | Best For |
|-----------|----------|----------|
| **LangGraph** | `neo4j_langgraph_mcp_agent.ipynb` | LangChain ecosystem users |
| **Strands Agents** | `neo4j_strands_mcp_agent.ipynb` | AWS-native, lightweight |

## LangGraph Approach

Uses LangChain's graph-based agent framework:
- Familiar to LangChain users
- Rich ecosystem of tools
- Flexible graph structure

## Strands Approach

Uses AWS's lightweight agent SDK:
- Minimal dependencies
- AWS-native integration
- Simple, direct approach

## Choose Based On

- Your existing stack
- Team familiarity
- Ecosystem preferences

Both produce equivalent results!
